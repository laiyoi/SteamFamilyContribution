# Steam 家庭库贡献率查询工具

> 由 vibe coding 制作

## 项目简介

Steam 家庭库贡献率查询工具是一个用于分析 Steam 家庭共享库中各成员贡献情况的命令行工具。它可以帮助家庭组管理员和成员了解每个人对家庭库的贡献比例，包括游戏数量和价值。

## 功能特点

- **多种登录方式**：支持用户名密码登录、Access Token 登录和扫码登录
- **完整的家庭组信息获取**：获取家庭组名称、成员列表及其角色
- **游戏库分析**：统计共享库中的游戏数量和总价值
- **贡献计算**：计算每个成员的贡献金额和比率（支持去重和不去重两种计算方式）
- **结果保存与分析**：将查询结果保存为 JSON 文件，支持后续分析
- **多地区价格支持**：可配置不同国家/地区的货币单位

## 技术栈

- C#
- .NET 6.0+
- SteamKit2

## 使用方法

### 首次运行

1. 下载并解压最新版本的工具
2. 运行 `SteamContribution.exe`
3. 首次运行时，工具会自动创建配置文件 `config.json`
4. 编辑配置文件或直接在命令行中输入用户名密码登录

### 配置文件说明

配置文件 `config.json` 包含以下选项：

```json
{
  "Username": "your_steam_username",
  "Password": "your_steam_password",
  "ApiKey": "your_steam_api_key",
  "CountryCode": "CN"
}
```

- `Username`：Steam 用户名（可选，可在登录时手动输入）
- `Password`：Steam 密码（可选，可在登录时手动输入）
- `ApiKey`：Steam API Key（可选，用于获取游戏价格信息）
- `CountryCode`：国家/地区代码，用于确定货币单位（默认为 "CN"）

### 操作流程

1. 运行工具后，选择操作模式：
   - `1. 完整功能（登录并查询家庭组）`：执行完整的查询流程
   - `2. 分析已保存的查询结果`：分析之前保存的查询结果

2. 选择登录方式：
   - `1. 用户名密码登录`：使用用户名和密码登录
   - `2. 使用已保存的 Access Token`：使用之前登录时保存的 Access Token
   - `3. 扫码登录`：通过手机 Steam 客户端扫码登录

3. 工具会自动执行以下操作：
   - 登录 Steam 账号
   - 获取家庭组信息
   - 获取共享库游戏列表
   - 查询游戏价格
   - 计算每个成员的贡献
   - 显示贡献统计汇总
   - 保存查询结果

4. 查询完成后，工具会显示每个成员的贡献情况，包括：
   - 贡献金额
   - 贡献比率（不去重和去重两种方式）
   - 游戏数量

### 结果文件

查询结果会保存到 `results` 文件夹中，文件名格式为 `query_result_yyyyMMdd_HHmmss.json`。

## 注意事项

- 本工具仅用于个人使用，请勿用于商业用途
- 请妥善保管您的 Steam 账号信息，工具会在本地保存登录凭证
- 频繁登录可能会触发 Steam 的安全措施，如验证码或限制登录
- 部分游戏可能无法获取价格信息，这会影响贡献计算的准确性

## 常见问题

### 登录失败

- 检查用户名和密码是否正确
- 检查网络连接是否正常
- 如果启用了 Steam 令牌，需要输入验证码
- 尝试使用扫码登录方式

### 无法获取家庭组信息

- 确保您已加入 Steam 家庭组
- 确保您有足够的权限查看家庭组信息

### 游戏价格查询失败

- 检查您的网络连接是否正常
- 尝试更换 `CountryCode` 配置
- 部分免费游戏可能无法获取价格信息
