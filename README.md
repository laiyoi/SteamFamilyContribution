# Steam 家庭库贡献率查询工具

基于 Steam Web API 开发的命令行工具，用于查询 Steam 用户游戏库并计算家庭成员的游戏贡献率。

## 功能特性

- ✅ **无需登录 Steam 账号** - 仅需 Steam Web API Key
- ✅ **自动检测地区** - 根据 IP 地址自动识别所在区域
- ✅ **自动获取家庭组** - 支持使用 Access Token 自动获取家庭成员列表（可选）
- ✅ **批量输入** - 支持一次输入多个 SteamID
- ✅ 查询用户游戏库
- ✅ 获取游戏区域价格（支持全球主要区域）
- ✅ 计算家庭库总价值
- ✅ 计算各成员贡献率
- ✅ 显示最值钱的游戏排行
- ✅ 重复游戏统计，帮助节省开支

## 前置要求

1. **.NET 10.0 SDK** 或更高版本
2. **Steam Web API Key**（免费获取）
   - 获取地址：https://steamcommunity.com/dev/apikey

## 快速开始

### 1. 获取 Steam Web API Key

1. 访问 https://steamcommunity.com/dev/apikey
2. 登录你的 Steam 账号
3. 输入任意域名（如 `localhost`）
4. 点击"注册"获取 API Key

### 2. 构建项目

```bash
dotnet build
```

### 3. 运行程序

```bash
dotnet run
```

### 4. 按提示输入信息

程序会自动：
1. ✅ 检测您所在的地区
2. ⏳ 输入 Steam Web API Key
3. ⏳ （可选）输入 Access Token 以自动获取家庭组成员
4. ⏳ 输入家庭成员的 SteamID（支持批量输入，如果已自动获取则跳过）

## 输出说明

### 贡献率计算

- **贡献价值**：该成员拥有的游戏在当前区域的总价格
- **贡献率**：该成员贡献价值占家庭库总价值的百分比
- 按贡献值从高到低排序显示

### 最值钱的游戏

- 显示家庭库中价格最高的 20 款游戏
- 标注每款游戏的拥有者数量
- 帮助了解家庭库的核心价值

### 重复游戏统计

- 统计被多个成员拥有的游戏
- 计算重复游戏的总价值
- 计算潜在节省金额（如果只购买一次）
- 提供购买建议，帮助节省开支

## Access Token 获取说明

Access Token 用于调用 Steam 家庭组 API，有以下几种获取方式：

### 方法 1：通过 Steam 登录获取
1. 使用 Steam 账号登录网页版 Steam
2. 打开浏览器开发者工具（F12）
3. 在 Network 标签页中找到任意 Steam API 请求
4. 从请求头中复制 `Authorization` 字段中的 token

### 方法 2：使用 SteamKit2 获取
```csharp
var client = new SteamClient();
var steamUser = client.GetHandler<SteamUser>();
// 登录后从回调中获取 access_token
```

### 方法 3：使用 Steam 移动端
1. 安装 Steam 手机应用
2. 使用抓包工具获取 API 请求中的 access_token

**注意**：Access Token 有有效期，过期需要重新获取。如果不提供 Access Token，程序将使用手动输入模式。

## 注意事项

1. **隐私设置**：目标账号的游戏库必须设置为公开
   - 在 Steam 客户端：设置 > 隐私设置 > 游戏详情 > 设为公开
   - 或者在网页版：编辑个人资料 > 隐私设置 > 游戏详情 > 公开

2. **API 限制**：Steam Web API 有调用频率限制
   - 程序已内置并发控制（最多 5 个并发请求）
   - 大量查询时可能被限流，请耐心等待

3. **价格数据**：
   - 免费游戏价格为 0
   - 部分游戏可能无价格信息（如早期测试游戏）
   - 价格以指定区域商店为准

4. **家庭共享说明**：
   - 本工具查询的是各成员各自拥有的游戏
   - 不是查询 Steam 家庭共享库中的游戏
   - 重复游戏统计可以帮助优化购买决策

5. **家庭组 API**：
   - 需要提供有效的 Access Token
   - Access Token 有有效期，过期需重新获取
   - 如果获取失败，程序会自动切换到手动输入模式

## 常见问题

### Q: 无法获取游戏库？
A: 
1. 确保提供了有效的 Steam Web API Key
2. 检查目标账号的游戏库隐私设置是否为公开
3. 确认 SteamID 格式正确（64 位，以 7656119 开头）

### Q: 贡献率计算不准确？
A: 
1. 确保所有家庭成员的游戏库都是公开状态
2. 提供了正确的 SteamID
3. 检查地区选择是否正确

### Q: 如何获取 API Key？
A: 
1. 访问 https://steamcommunity.com/dev/apikey
2. 登录 Steam 账号
3. 同意条款并注册
4. 复制显示的 API Key

### Q: 为什么检测的地区不准确？
A: 
- 地区检测基于 IP 地址，如果使用代理/VPN 可能检测不准确
- 可以手动修改代码中的默认地区

### Q: 支持查询多少个家庭成员？
A: 
- 理论上没有数量限制
- 但成员过多会导致查询时间较长
- 建议一次查询 2-10 个成员

## 技术架构

- **HttpClient**：调用 Steam Web API 和 IP 地址 API
- **System.Text.Json**：JSON 数据解析
- **并发控制**：使用 SemaphoreSlim 限制 API 并发请求数
- **LINQ**：数据查询和统计
- **SteamKit2**：备用地区检测（CellID）
